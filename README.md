# GitHub Final Project – Simple Interest Calculator

This repository is part of the GitHub final project assignment. It demonstrates Git and GitHub basics by building a **Simple Interest Calculator** using Bash scripting.

## 📌 Project Overview
The calculator computes **Simple Interest (SI)** based on user input values:
- **Principal (P)** – the initial amount of money
- **Rate of Interest (R)** – the annual interest rate (percentage)
- **Time (T)** – the time period in years

Formula used:


\[
SI = \frac{P \times R \times T}{100}
\]
